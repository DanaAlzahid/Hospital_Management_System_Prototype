## Overview
This project is a simple Hospital Management System (HMS) prototype built using **C# WinForms** on **.NET 6**.  
It demonstrates core functionalities such as:

- Patient management  
- Viewing medical records  
- Medication prescription (doctor)  
- AI-based diet recommendation module  
- Viewing diet plans  

The AI recommender is **simulated using predefined rule-based logic** for demonstration purposes.

---

## How to Run the Application

### Requirements
- Install **.NET 6 SDK**  
- Install **Visual Studio 2022/2023**  
  *(or Visual Studio Code with C# extensions)*

### Steps
1. Open the solution file: **HMS_AI_Prototype.sln** in Visual Studio.  
2. Build the project.  
3. Run the application.  
4. Log in using the demo credentials:

   **Email:** admin@hms.com  
   **Password:** 1234  

   *These are the default demo credentials used to access the prototype when running the application.*
